# 3D Perspective Glitch Hover Effect

A fun hover effect that shows a pixelated image with a glitch animation as seen on [bu≈çy's website](https://buoy.work/gallery/).

## Installation

Install dependencies:

```
npm install
```

Compile the code for development and start a local server:

```
npm start
```

Create the build:

```
npm run build
```
